## Page speed measurement tool (branch: 2026-06-page-speed)

- Created `tests/perf/measure-vitals.js` — Playwright Core Web Vitals reporter.
  - Measures TTFB, FCP, LCP, CLS, TBT, DOM Load, Page Load, doc transfer size, and sub-resource weight (JS/CSS/img breakdown).
  - Each run uses a fresh uncached browser context (simulates cold first visit).
  - N runs per page, reports medians. Default N=3.
  - Optional Google PageSpeed Insights integration via `PAGESPEED_API_KEY` (production only).
  - Parameterised via env vars: `BASE_URL`, `PAGESPEED_API_KEY`, `PERF_RUNS`, `OUTPUT_JSON`.
- Added `docker compose run perf` service (manual profile, reuses `tests` Docker target).
- Added `npm run test:perf` script.
- Added `.env.example`; `.env` and `perf-report*.json` gitignored.

## Local validation workflow

Script: `.github/scripts/validate-local-build.sh`

Run after `docker compose --profile manual up -d http_serve`:
```bash
.github/scripts/validate-local-build.sh
```

Steps it runs:
1. Checks http_serve container is running (port 8082)
2. HTTP smoke checks for /, /construction_guide.html, /tags/arduino/
3. Counts AVIF/WebP files in _site/assets/images (verifies image build)
4. BDD tests via `docker run` on the `orionrobotsgithubio_default` network pointing BASE_URL at the container

**Known limitation**: Gallery redirect test (scenario 5) fails locally — htaccess rewrites not active in http_serve. Issue: #413. Script warns but does not fail hard on this.

BDD test image: `orionrobotsgithubio-test`
Docker network: `orionrobotsgithubio_default`
