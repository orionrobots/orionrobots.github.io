Welcome to the future...


In a field like this, where research and a lot of planning is important, the future is always difficult to pin down. This
section intends to display projects that I have yet to start work on. In time, these will be moved
away and into other sections. I hope to inspire other robot builders and maybe encourage suggestions for new projects from

Lego CAD tool improvements

Since I have suggested them, maybe sometime in the future I can add engineering and electronic simulation to the Lego CAD tools, as well as a few other improvements.

Of course I fully intend to make my results Open source, and GPL them. I will also post them here.

This will include an RCX simulator, which can interpret NQC and other RCX code file formats. Along with the ability to add multiple RCX's and even use graphs to simulate custom add ons.

The other option is to use some logic similar to the great FrameSticks or Darwin2k projects(I am still yet to explore the latter), and allow the model file, along with imported RCX programs to function as a genotype - which can be evolved in a genetic algorithm. Things like adding part constrains(fewest parts, lowest battery consumption, only use standard mindstorms set, or additional sets-mars explorer) could evolve interesting buildable units. Also things like beam stresses could be used to make sure it is not going to break beams, burn motor coils and de-tooth sprockets would be useful.

One day I hope to have time to realise this fully. In the meantime, if any of the visitors do, then please let me know on the feedback page. Links are always welcome here
