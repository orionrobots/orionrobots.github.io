This is sometimes used as part of processes to build robots.

It is used in the manufacture of PCB's when use photoresist. It is also used to clear older EPROM chips, and can still be used to do the same with EEPROM chips.

It is not normally used in sensor technology - as it is considered more harmful or intrusive than IR. However - it can be used to reveal things not normally visible in ambient light. It is not difficult to see a robot or machine using UV to check the validity of monatary notes.
